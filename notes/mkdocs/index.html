    mkdocs 支持无序列表嵌套的方法

    安装 mdx_truly_sane_lists

    pip install mdx_truly_sane_lists

    在 mkdoc.yml 中添加

    markdown_extensions:
    - mdx_truly_sane_lists

    mkdocs 支持 mermaid 图表功能

    参考 Markdown 的项目文档 插件中的图像，表格，图表和图形

    安装 mkdocs-mermaid2-plugin

    pip install mkdocs-mermaid2-plugin

    在 mkdoc.yml 中添加

    plugins:
    - search
    - mermaid2
